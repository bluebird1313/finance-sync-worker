Finance Bot Setup Guide

Step 1: Create a Slack App

  1. Go to https://api.slack.com/apps
  2. Click "Create New App" and choose "From scratch"
  3. Enter "Finance Bot" as the app name
  4. Select your workspace and click "Create App"

Step 2: Configure Slash Command

  1. In the sidebar, select "Slash Commands" under "Features"
  2. Click "Create New Command"
  3. Fill in the following details:
    • Command: /finance
    • Request URL: https://your-worker-url.workers.dev/slack/command
    • Short Description: "Query financial data"
    • Usage Hint: "[query]"
  4. Click "Save"

Step 3: Set Up Permissions

  1. In the sidebar, select "OAuth & Permissions" under "Features"
  2. Under "Scopes", add the following Bot Token Scopes:
    • commands - To create the slash command
    • chat:write - To send messages to channels
  3. Scroll up to "OAuth Tokens for Your Workspace" and click "Install to Workspace"
  4. Review the permissions and click "Allow"
  5. Copy the "Bot User OAuth Token" for later use

Step 4: Configure Incoming Webhooks

  1. In the sidebar, select "Incoming Webhooks" under "Features"
  2. Toggle "Activate Incoming Webhooks" to On
  3. Click "Add New Webhook to Workspace"
  4. Select the channel where you want to receive financial alerts
  5. Click "Allow"
  6. Copy the Webhook URL for use in your environment variables

Step 5: Update Environment Variables

# Add to your .env file
SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...
SLACK_VERIFICATION_TOKEN=your-verification-token

Find your verification token in the "Basic Information" section under "App Credentials".

Example Queries

Here are some example queries you can try with the /finance slash command:

  • /finance What was our revenue last month?
  • /finance Show me monthly expenses for the past year
  • /finance What's our current profit margin?
  • /finance Show me recent transactions
  • /finance What are our account balances?
